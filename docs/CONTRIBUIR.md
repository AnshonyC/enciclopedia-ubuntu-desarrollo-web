# Contribuir

Gracias por querer mejorar este manual.

Este proyecto busca ayudar a personas que están empezando en Linux/Ubuntu/Kubuntu y desarrollo web local con PHP puro, Apache, MariaDB, Git y SSH.

## Formas de ayudar

Puedes ayudar de estas maneras:

- Reportando errores.
- Sugiriendo comandos más seguros.
- Mejorando explicaciones.
- Agregando secciones faltantes.
- Corrigiendo ortografía.
- Probando el manual en otras versiones de Ubuntu/Kubuntu.
- Compartiendo problemas reales y cómo los resolviste.

## Antes de proponer cambios

Por favor revisa que:

- El comando sea seguro para principiantes.
- La explicación sea clara.
- No se incluyan contraseñas reales.
- No se incluyan rutas personales.
- No se agreguen datos privados.
- No se recomienden comandos destructivos sin advertencia.

## Pull Requests

Puedes hacer un fork del repositorio, crear una rama y enviar un Pull Request.

Ejemplo de ramas:

- `fix/comando-apache`
- `docs/seccion-ssh`
- `ui/mejora-busqueda`
- `security/advertencias`

## Issues

Al abrir un Issue, intenta incluir:

- Qué estabas intentando hacer.
- Qué sistema usas.
- Qué comando ejecutaste.
- Qué error apareció.
- Qué esperabas que pasara.

# Créditos y uso de IA

Este proyecto fue creado, ordenado y ampliado con ayuda de inteligencia artificial.

La IA se usó como apoyo para:

- Organizar secciones.
- Mejorar explicaciones.
- Proponer comandos.
- Crear estructura del repositorio.
- Redactar documentación.
- Mejorar la interfaz del archivo HTML.

Aun así, el contenido debe revisarse y probarse. Puede haber errores, comandos que cambien según la distribución, casos no cubiertos o recomendaciones que necesiten ajuste.

Si encuentras algo incorrecto, puedes abrir un Issue o enviar un Pull Request.
